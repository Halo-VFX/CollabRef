CollabRef - Collaborative Reference Board
==========================================

Just run CollabRef.exe to start

Requirements:
- Windows 10 or later (64-bit)
- Visual C++ Redistributable 2022
  Download: https://aka.ms/vs/17/release/vc_redist.x64.exe

Controls:
- Right-click: Menu
- Middle-click or Space+drag: Pan
- Scroll wheel: Zoom
- Drag and drop images onto window
- Ctrl+V: Paste image from clipboard

Collaboration:
1. Right-click > Collaborate > Host Session
2. Share the Room ID with others
3. Others: Join Session with same Room ID
